# Friends of Lake Henry

This repository contains the website for **Friends of Lake Henry**, a nonprofit organization dedicated to raising funds and awareness to support the restoration and preservation of Lake Henry.

## Purpose
The site provides:
- Information about the lake and restoration efforts
- Community event listings
- Raffle and fundraising information
- Donation and contact details

## Technology
- Astro (static site generator)
- Cloudflare Pages for hosting and CI/CD
- Content is managed separately to allow non-technical editors to update events and raffle information

## Development
This site is deployed automatically via Cloudflare Pages when changes are pushed to the main branch.
